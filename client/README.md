# Server 1586 - React Frontend with HeroUI v3

This is the modernized React frontend for Server 1586, migrated from vanilla HTML/CSS/JS to React with HeroUI v3 components.

## Features

✅ **Modern Stack**
- React 18 with TypeScript
- Vite for blazing-fast development
- HeroUI v3 (Alpha) - Modern, accessible component library
- Tailwind CSS v4 for utility-first styling

✅ **UI/UX Improvements**
- WCAG AA compliant text contrast (fixes original #888 gray issues)
- Built-in light/dark mode with persistent theme selection
- Responsive design for mobile, tablet, and desktop
- Smooth animations and transitions
- Accessible navigation with keyboard support

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Visit **http://localhost:5173/** to see the app.
