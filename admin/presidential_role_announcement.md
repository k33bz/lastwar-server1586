# Presidential Role Discord Announcement

## Option 1: Focused Version (President Role Only)

```
🎖️ **NEW PRESIDENTIAL ROLE ANNOUNCEMENT** 🎖️

Server 1586 has introduced a new **President** role on our website admin panel!

**🌐 Website:** https://www.lastwar1586.online

**What is the President Role?**
The President role is designed for alliance leaders and trusted server representatives who need access to key management features on the Server 1586 website. This role sits between R5 and Admin, providing important capabilities while maintaining appropriate access controls.

**🗳️ Council Rotation Management** ⭐ NEW FEATURE!
Presidents can now regenerate the alliance voting schedule directly from the website admin panel!

**How to Access:**
1. Visit https://www.lastwar1586.online
2. Log in with your President account credentials
3. Navigate to **Alliances → 🗳️ Council Rotation**

**What You Can Do:**
• **View Status:** See current week, next rotation date, and rotating pool
• **Regenerate Schedule:** Recalculate future weeks when rankings change
• **Fair Distribution:** Algorithm ensures all rotating alliances (ranks 6-15) get equal representation
• **Protected History:** Past and current weeks are never modified, only future weeks are regenerated

**Step-by-Step:**
1. Navigate to the Council Rotation page on the website
2. Review current status and statistics
3. Click "🔄 Regenerate Future Weeks" when needed
4. Confirm the regeneration
5. View fairness distribution results

**When to Regenerate:**
✅ Alliance rankings change significantly
✅ New alliances enter top 15
✅ Alliances leave top 15
✅ Manual fairness adjustment needed

**Additional President Privileges on the Website:**
• Access to Season 2 event management and announcements (R4+ feature)
• Discord message template creation and management (R4+ feature)
• Alliance data viewing and coordination tools
• View and manage server event calendars

**Security:**
All actions taken by Presidents are logged in the audit system for transparency and accountability. The role is designed to empower trusted leaders while maintaining proper oversight.

**Need Access?**
If you're an alliance leader or trusted representative and need President access to the website, please contact a Server Admin.

**🌐 Website:** https://www.lastwar1586.online

— Server 1586 Administration Team
```
