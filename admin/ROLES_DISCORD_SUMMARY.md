# Server 1586 Website Roles - Discord Summary

## Quick Summary for Discord

```
📋 **SERVER 1586 WEBSITE ROLES EXPLAINED**

**🌐 Website:** https://www.lastwar1586.online

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🔴 ADMIN** (Super Administrator)
The highest access level with full system control.

**What Admins Can Do:**
✅ Everything (full system access)
✅ Manage all user accounts and roles
✅ Configure Discord bot and Season 2 settings
✅ Alliance power editor and bulk tools
✅ Backup/restore system access
✅ Delete anyone's messages
✅ Approve global templates
✅ View all audit logs

**Who Gets This:** Server administrators and technical managers only.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🟡 PRESIDENT** (Trusted Leadership)
Elevated access for server leaders and coordinators.

**What Presidents Can Do:**
✅ **Regenerate council rotation schedule** ⭐ EXCLUSIVE!
✅ Send Discord messages (instant, scheduled, recurring)
✅ Auto-delete messages (1h-48h options)
✅ Create and manage message templates
✅ View and announce Season 2 events
✅ View alliance data and rankings

**What Presidents CANNOT Do:**
❌ Create/delete user accounts
❌ Configure Discord or Season 2 settings
❌ Access backup/restore
❌ Delete others' messages
❌ Modify alliance data

**Who Gets This:** Top alliance leaders, council representatives, and trusted server coordinators.

**Key Feature - Council Rotation:**
Presidents can regenerate the alliance voting schedule when rankings change, ensuring fair distribution of rotating council seats (ranks 6-15).

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🟢 R5** (Alliance Leader)
Alliance leadership with communication tools.

**What R5 Can Do:**
✅ Send Discord messages to alliance channels
✅ Schedule and create recurring messages
✅ Auto-delete messages (1h-48h)
✅ Create message templates
✅ View and announce Season 2 events
✅ Edit/delete their own messages only

**What R5 CANNOT Do:**
❌ Regenerate council rotation (President/Admin only)
❌ Configure system settings
❌ Delete others' messages
❌ Access other alliances' channels

**Who Gets This:** Alliance leaders (R5 rank in-game).

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🔵 R4** (Alliance Officer)
Alliance officer with same access as R5.

**What R4 Can Do:**
✅ Send Discord messages to alliance channels
✅ Schedule and create recurring messages
✅ Auto-delete messages (1h-48h)
✅ Create message templates
✅ View and announce Season 2 events
✅ Edit/delete their own messages only

**Same as R5!** R4 and R5 have identical website permissions.

**Who Gets This:** Alliance officers (R4 rank in-game).

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🔷 R3** (Alliance Member)
Basic member access.

**What R3 Can Do:**
✅ View public server information
✅ Access personal profile

**Who Gets This:** Regular alliance members.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**QUICK COMPARISON:**

**Council Rotation:**
🔴 Admin ✅ | 🟡 President ✅ | 🟢 R5 ❌ | 🔵 R4 ❌

**Discord Messages:**
🔴 Admin ✅ | 🟡 President ✅ | 🟢 R5 ✅ | 🔵 R4 ✅

**Season 2 Events:**
🔴 Admin ✅ | 🟡 President ✅ | 🟢 R5 ✅ | 🔵 R4 ✅

**System Configuration:**
🔴 Admin ✅ | 🟡 President ❌ | 🟢 R5 ❌ | 🔵 R4 ❌

**Delete Others' Messages:**
🔴 Admin ✅ | 🟡 President ❌ | 🟢 R5 ❌ | 🔵 R4 ❌

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**KEY POINTS:**

1️⃣ **President = R5/R4 + Council Rotation Access**
Presidents have all the same Discord and Season 2 features as R5/R4, PLUS the ability to regenerate the council rotation schedule.

2️⃣ **R5 and R4 are Identical**
Both alliance leaders and officers have the same website permissions. The distinction is organizational only.

3️⃣ **Only Admin Can Configure**
System settings, user management, and Discord/Season 2 configuration require Admin access.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**NEED ACCESS?**
Contact a Server Admin to request website access based on your in-game rank and alliance.

**Website:** https://www.lastwar1586.online

## Simplified One-Liner Descriptions

```
**Role Descriptions in One Line:**

🔴 **Admin:** Full system control - manages everything including users, settings, and configurations

🟡 **President:** Trusted leadership - can regenerate council rotation + all R5/R4 features

🟢 **R5:** Alliance leader - sends Discord messages, manages events, coordinates alliance

🔵 **R4:** Alliance officer - same as R5, assists with alliance communications

🔷 **R3:** Alliance member - basic viewing access only
