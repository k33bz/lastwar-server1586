# OCR Work - Future Development

This folder contains OCR (Optical Character Recognition) related work for processing Last War alliance screenshots.

## Contents

- `process-screenshots-easyocr.py` - EasyOCR-based screenshot processor
- `process-screenshots-v3.py` - Alternative OCR processor (v3)
- `debug_ocr/` - OCR debugging files and test data
- `admin/` - Admin-related OCR work
- `alliance_cards/` - Alliance card screenshot processing

## Status

**⚠️ Work in Progress** - This functionality is not yet integrated into the main website.

## Future Plans

- Extract alliance data from screenshots automatically
- Process power rankings from game screenshots
- Auto-update alliance.json from screenshot data

## Usage

Not yet ready for production use. Scripts require additional dependencies:
- EasyOCR
- Tesseract OCR
- Additional Python packages
