# Product Overview

Server 1586 is a comprehensive web application for managing Last War alliance rankings, council voting, and server rules for Server 1586.

## Core Features

- **Alliance Rankings**: Display top 15 alliances with dynamic power-based ranking
- **Council Voting System**: Rotating council members (ranks 6-15) with automatic weekly rotation
- **Server Rules Management**: Collapsible rules with amendment history tracking
- **Admin Panel**: JWT-based authentication system for alliance and user management
- **OCR Integration**: Screenshot processing for automated alliance data extraction

## Target Users

- **Alliance Leaders (R5)**: Manage alliance data, sign server rules
- **Alliance Officers (R4)**: Edit alliance information
- **Power Editors (APE)**: Special permission for alliance power management
- **Server Administrators**: Full system access and user management
- **Server Members**: View rankings, rules, and council information

## Key Value Propositions

- Automated council rotation with fair distribution algorithm
- Real-time alliance power tracking and visualization
- Secure passwordless authentication via magic links
- Comprehensive audit logging and security monitoring
- Mobile-responsive design for all devices
